# Images Folder

Upload your photos here! This folder contains all images used in your portfolio.

## Folder Structure

### `/images/projects/`
Upload project screenshots/images here. Name them:
- `project-1.jpg` (or .png)
- `project-2.jpg`
- `project-3.jpg`
- etc.

### `/images/about/`
Upload your profile photo here:
- `profile.jpg` (or .png)

### `/images/playground/`
Upload playground/experiment images here if needed.

## Image Guidelines

- **Recommended sizes:**
  - Project images: 800x600px or larger (16:9 aspect ratio works best)
  - Profile photo: 600x600px (square)
  
- **File formats:** JPG, PNG, or WebP
- **File size:** Try to keep images under 500KB for faster loading

## How to Use

1. Upload your images to the appropriate folders
2. The components will automatically use these images
3. If an image doesn't exist, placeholder images will be shown
